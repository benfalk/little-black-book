== README

This is a rails web service application that provides contact
information. It is meant to connect to different data sources
to aggregate that information into a single area.

== REQUIREMENTS

* Ruby >= 2.0
* Rails >= 4.0

== DEVELOPMENT INFORMATION

You can seed 10 sample contacts in the database with the following rake comand:

  rake sample:contacts

The test suite can be run with the following rake command:

  rake rspec
